# Price-MA Deviation (PMAD)

Price-MA Deviation (價格-均線偏離指標, PMAD) 是一個 TradingView Pine Script 指標，旨在量化價格相對於其移動平均線 (Moving Average) 的「過度延伸」程度。它將這種偏離標準化到 0-100 的範圍內，使其成為識別在不同時間框架內市場是否出現過度延伸狀況的強大工具。

## 為什麼要使用 PMAD？

在趨勢市場中，價格與其均值（移動平均線）之間的距離會不斷波動。單純看「偏離 MA 的百分比」並不能提供太多資訊，因為市場的波動性是不斷變化的。

PMAD 的解決方案是**根據其自身的歷史範圍來標準化這個偏離值**：

- 它不只是告訴你價格偏離了 MA 5%。
- 它告訴你，5% 的偏離是否為過去 200 根 K 線中的最高點（數值 100），或者僅僅是一個普通的波動（數值 50）。

## 核心邏輯

1.  **選擇 (Selection):** 選擇你想計算的價格來源（預設為 `hlc3`，即典型價格）、移動平均線的類型 (SMA/EMA) 以及快線 (Fast) 與慢線 (Slow) 的週期長度。
2.  **計算 (Calculation):** 計算原始的百分比差異：`(價格 - MA) / MA * 100`。
3.  **標準化 (Normalization):** 它會回顧一個由使用者定義的區間（`標準化回顧週期`），找出這些偏離值的歷史高點/低點，並將當前值映射到 0-100 的範圍內。
 
簡單來說就是：

```text
原始偏離值 = (價格 - 移動平均線) / 移動平均線 * 100
標準化的偏離值 = 當前的原始偏離值在其最近的最低-最高範圍內所處的位置
```

## 如何解讀雙線系統

PMAD 的優勢在於**快線偏離** (Fast Deviation，粗線) 與**慢線偏離** (Slow Deviation，細線) 之間的相互作用。

### 收斂與發散的力量
- **宏觀-微觀對齊 (Macro-Micro Alignment):** 當慢線和快線同時觸及極端區域（大於 85 或小於 15）時，意味著市場在多個時間框架上都達到了「歷史性的過度延伸」。這些是均值回歸 (mean reversion) 或趨勢顯著耗盡的高機率區域。
- **長趨勢中的短期回檔 (Short-term Pullbacks in Long-term Trends):** 如果慢線穩定保持在 50-60 左右（健康的上升趨勢中），而快線跌破 15，這通常表示在強勁趨勢中出現了一個「逢低買進 (buy the dip)」的機會，而不是趨勢反轉。
- **標準化背離 (Normalization Divergence):** 如果價格創下新高，但 PMAD 快線達到的峰值卻低於前一個峰值，這表示價格上推的動能正在減弱（頂背離）——這通常是趨勢轉變的前兆。

### 理解數值範圍

以下是關鍵的 PMAD 數值範圍與預設配置（快線：20 EMA / 150 回顧週期 | 慢線：200 SMA / 200 回顧週期）的快速參考：

| 數值級別     | 區域結構         | 市場情緒                                     |
| :----------- | :--------------- | :------------------------------------------- |
| **85 - 100** | **頂部過度延伸** | 極端的看漲延伸。「橡皮筋」已被拉到極限。     |
| **50**       | **中點平衡區**   | 價格在其 MA 周圍正常震盪。市場處於均衡狀態。 |
| **0 - 15**   | **底部過度延伸** | 極端的看跌延伸。近期出現恐慌性拋售或高潮。   |

建議將 PMAD 作為判斷動能與波動率的工具（或是衡量相對延伸度的工具），而非單獨的買賣訊號。一個極端的讀數（0 或 100）表示市場出現了延伸，而不是立即的反轉。市場保持在過度延伸狀態的時間，通常可以比大多數交易者維持資金流動性的時間還要長。請將本指標與價格走勢 (price action) 及成交量結合使用。
